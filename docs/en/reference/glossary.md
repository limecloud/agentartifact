# Glossary

| Term | Meaning |
| --- | --- |
| Agent Artifact | Portable standard for durable agent deliverables. |
| Artifact | Stable envelope for one deliverable across versions and exports. |
| Artifact kind | Domain meaning of a deliverable, separate from file extension. |
| Artifact part | Block, file, media part, page, slide, transcript segment, patch, table, or annotation. |
| Byte ref | Pointer to bytes with media type, size, digest, and access constraints. |
| Version | Immutable artifact snapshot. |
| Operation | Typed mutation intent or applied change. |
| Diff | Structured comparison between versions, parts, or bytes. |
| Source link | Relationship between artifact content and origins. |
| Render manifest | Preview and viewer contract for UI surfaces. |
| Export manifest | Record of exported package, files, integrity, redaction, and completeness. |
| Handoff package | Portable package for another system or peer agent. |

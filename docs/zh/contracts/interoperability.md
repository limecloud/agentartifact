# Interoperability

Agent Artifact 是 storage-neutral 与 UI-neutral 的。MCP resources、A2A artifacts、OCI descriptors、W3C PROV、Web Annotation、in-toto/SLSA、BagIt/RO-Crate、C2PA、CloudEvents 与 OpenTelemetry 都是参考，不转移语义所有权。

# Operations 与 diffs

operations 描述 artifact 如何变化，例如 create、add_part、update_part、remove_part、reorder_parts、attach_source、finalize_version、redact、export、handoff。diff 应识别 part、byte ref、source link、order、status 与 policy/export state 的变化。

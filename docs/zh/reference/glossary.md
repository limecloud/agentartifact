# 术语表

| Term | 含义 |
| --- | --- |
| Agent Artifact | 面向 durable agent deliverables 的可移植标准。 |
| Artifact | 一个交付物跨版本与导出的稳定 envelope。 |
| Artifact kind | 交付物的领域含义，区别于文件扩展名。 |
| Artifact part | Block、file、media part、page、slide、transcript segment、patch、table 或 annotation。 |
| Byte ref | 带 media type、size、digest 与 access constraints 的 bytes 指针。 |
| Version | 不可变 artifact snapshot。 |
| Operation | typed mutation intent 或 applied change。 |
| Diff | versions、parts 或 bytes 之间的结构化比较。 |
| Source link | artifact content 与 origins 的关系。 |
| Render manifest | UI surface 的 preview 与 viewer contract。 |
| Export manifest | exported package、files、integrity、redaction 与 completeness 记录。 |
| Handoff package | 给另一个系统或 peer agent 的可移植 package。 |

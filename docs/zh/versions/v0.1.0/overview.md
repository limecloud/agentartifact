# v0.1.0 概览

Agent Artifact v0.1.0 是第一个公开草案，定义 Agent outputs 的 durable deliverable layer。

## Included

- Artifact envelope、kind、status、ownership 与 refs。
- Artifact parts 与 byte refs。
- Versions、snapshots、operations 与 diffs。
- Source links、render manifests、export manifests 与 handoff packages。
- Event classes 与 JSON Schemas。
- 中英文文档。
- LLM-friendly `llms.txt` 与 `llms-full.txt`。
